# Modelisation_Covid19
Avec l’épidémie de Covid-19, nous constatons la difficulté que les épidémiologistes ont pour prédire l’évolution d’une pandémie. Cette difficulté se reflète sur les prises de décisions politiques qui pourtant requièrent une information fiable. L’objectif de ce projet est d’utiliser les données disponibles  pour aborder la question suivante : Peut-on mesurer rigoureusement l’influence d’un confinement sur la tendance épidémique ?
